{{"HELLO WORLD"}}

The paragraph above is wrapped in double curly braces. Double curly braces represent binding. To make this work, include angularjs in the header; and then add ng-app to the opening html tag.

The ng-model on the input above is first.greeting. first is an object; greeting is a property. The ng-class below is looking at ng-model to know what to show in the curly braces below.

{{first.greeting}} {{"World"}}

In addition, the ng-class attr above applies the value of first.greeting as a class to this div that's surrounding this element. This is a cool way to pull in bootstrap things [like if someone entered 'panel' or 'alert'].
